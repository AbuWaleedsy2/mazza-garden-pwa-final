# نشر التطبيق على Netlify (Drag & Drop)

## الطريقة الأسرع
1) فك ضغط الملف ZIP.
2) افتح Netlify → Add new site → Deploy manually.
3) اسحب **كل الملفات** داخل مجلد المشروع (بحيث ترى `index.html` و`sw.js` و`manifest.webmanifest` مباشرة) إلى منطقة الرفع. تأكد أن مجلد `icons/` وملفات `_redirects` و`404.html` ضمن الملفات المرفوعة.
4) بعد النشر سيظهر رابط. افتحه من الجوال للتجربة.

## إذا ظهر خطأ 404 عند التنقل
- تأكد من وجود ملف `_redirects` بجانب `index.html`.
- تأكد من وجود الملف `404.html` والذي يعيد التوجيه إلى الصفحة الرئيسية لمنع ظهور صفحة خطأ.
